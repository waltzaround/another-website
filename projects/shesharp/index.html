She#
Rebuilding a website for a charity serving the next generation of women in tech in 48 hours

Challenge
Modernize an old bootstrap website for a charity, provide a way to change the content without coding using a content management system like Wordpress, and drive more traffic to it.

Response
Designed a rough plan, did some information architecture, and some user needs research. Hacked a wordpress website together with PHP and CSS in a team of 12 to learn more about PHP, CSS and Wordpress in 48 hours.

Built as part of an ImpactNPO Hackathon.

Website: shesharp.co.nz

Skills: UX Design, Wordpress, PHP, CSS

Date: September 2017
