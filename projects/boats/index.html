NZTBS
Rebuilding a website for a traditional boatbuilding charity in 48 hours

Challenge
The charity was looking for a new website, as well as ways to automate how classes were organized for staff and students.

Response
Did some user research and gathered functional requirements from the directors of the school. Our team of 5 then hacked a wordpress website together with PHP and CSS from an existing theme to beat the 48hr hackathon deadline.

We even managed to consolidate all classes to automatically push to the website from eventbrite, eliminating any need to touch the website for day to day operations.

Built as part of an ImpactNPO Hackathon.

Website: nztbs.org.nz

Skills: UX Design, Wordpress, PHP, CSS

Date: April 2019
